# OSCControl_Unity

## C#脚本实现功能
1. 自动生成立方体阵列，并随机生成颜色，可参数调节生成的行数和列数
2. 摄像机自动定位到中央
3. 立方体阵列跟随移动
  * 跟随鼠标移动
  * 链接OSC并随着OSC的点按进行移动
